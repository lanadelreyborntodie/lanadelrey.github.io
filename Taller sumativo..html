"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploing content to the Internet and linking it to page created by other people, you become an active participant in the World Wide Web.

HTML uses "markup" to annotate text images, and other content for display in a Web browser. HTML markup includes special "elements" such as < head>, < title>, < body>, < header>, < footer>,
< article>, < section>, < p>, < div>, < span>, < ing>, < aside>, < audio>, < canvas>, < datalist>, < details>, < embed>, < nav>, < output>, < progress>, < video>, < ul>, < ol>, < li> and many others.

An HTML element is set off from other text in a document by "tags", which consist of the element name surrounded by "<" and ">". The name of an element inside a tag is charset insentive. That is, it can be written in uppercase, lowercase, or a mixture. For example, the < title> tag can be written as < Title> < TITLE> or in any other way.
